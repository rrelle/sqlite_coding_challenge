# SQLite Analytics Challenge

## Overview
This project uses SQL queries to analyze a sample business database. The goal was to answer key business questions using joins, aggregation, and filtering.

## Tools Used
- Visual Studio Code
- SQLite Extension

## Tasks Completed

### Task 1: Top 5 Customers by Total Spend
Identified the five customers who spent the most overall by summing order item totals.

### Task 2: Revenue by Product Category
Calculated total revenue for each product category to see which categories perform best.

### Task 3: Employees Above Department Average
Found employees whose salary is higher than the average salary in their department.

### Task 4: Cities with the Most Loyal Customers
Ranked cities based on the number of Gold-level loyalty customers.

## Key Insights
- A small number of customers contribute a large portion of total revenue.
- Certain product categories generate significantly more revenue than others.
- Some employees earn noticeably more than their department average.
- Gold customers are more concentrated in specific cities.

## Files Included
- `challenge.sql` → Contains all SQL queries for the tasks
- `INSIGHTS.md` → Summary of findings and observations
